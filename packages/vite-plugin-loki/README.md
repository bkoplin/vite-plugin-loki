PROJECT TAGLINE HERE

## Installation

```bash
pnpm i -D @bkoplin/vite-plugin-loki
```

## Usage

### add to vite config

```ts
// vite config
import { defineConfig } from 'vite';
import { loki } from '@bkoplin/vite-plugin-loki';

export default defineConfig({
	plugins: [
		loki({
			/* plugin options */
		})
	]
});
```
